## 一、【重点】6 个状态区分度重做

### 问题
现在每个角色的 6 个状态（`idle`/`thinking`/`working`/`attention`/`done`/`error`）**看起来太像**——
基本只换了眼睛/小点缀，整体轮廓、姿态、配色几乎不变。在顶栏 ~20px 的小尺寸下，
用户**一眼分不出宠物现在是「在思考」还是「在干活」还是「出错了」**，状态机的意义就削弱了。

### 目标
**每个状态在 16–22px 下都能被瞬间区分**——不用盯着看细节，扫一眼就知道宠物现在什么状态。

### 怎么做（请加大状态间差异，别只改眼睛）
对每个状态，请至少叠加 **2 种以上**强区分手段，而不是只改表情：

| 区分手段 | 说明 |
|---|---|
| **整体色调 / 光晕** | 给每个状态一个主色倾向：如 working 暖黄活力、attention 红、error 灰暗/偏冷、done 多彩、thinking 中性蓝、idle 低饱和。可用身体描边色、背景小光晕或腮红色体现。 |
| **姿态 / 朝向** | 改变重心或倾斜：idle 下沉/歪头打盹、working 前倾有干劲、attention 挺直惊起、error 瘫软/倒向一边。 |
| **大点缀（占比要大）** | 点缀别太小：idle 大 Zzz、thinking 明显思考气泡、working 醒目火花/齿轮、attention 大感叹号、done 满屏闪光彩带、error 大汗滴 + ×× 眼。 |
| **眼睛/嘴形** | 仍要做，但作为**辅助**而非唯一区分。 |

### 各状态目标印象（请强化到「一眼可辨」）

| 状态 | 一眼应读出 | 强区分建议 |
|---|---|---|
| `idle` | 「在睡/发呆」 | 眼闭、身体下沉或歪倒、大 Zzz、低饱和度 |
| `thinking` | 「在动脑」 | 眼上瞟、头顶明显思考气泡/问号、偏蓝中性调 |
| `working` | 「忙着干活」 | 前倾有动势、醒目火花/齿轮/动感线、暖黄活力色 |
| `attention` | 「快看我！」 | 惊讶瞪眼张嘴、身体挺起、大红感叹号、整体偏红 |
| `done` | 「成功啦」 | 大笑/星星眼、庆祝姿态、满屏彩色闪光彩带 |
| `error` | 「出错了/糟糕」 | ×× 眼或眩晕、瘫软/倒向、大汗滴、灰暗或冷色调 |

> 关键：**同一角色 6 张图摆一起，应该像 6 个不同情绪的瞬间，而不是「同一张脸换了下眼睛」。**
> 但仍是**同一个角色**（保持其招牌轮廓/配色基底/记忆点），只是情绪与姿态拉开差距。

### 交付
覆盖现有 `src/icons/mascots/<id>/{idle,thinking,working,attention,done,error}.svg`（5 角色 × 6 = 30 个）。
（一轮已交付的 `thinking` 情绪变体见 `DESIGN_SPEC.md` 第七节，可一并按新区分度风格做。）

## 二、【新增资产】像素「账号头像」图标

### 用途
点击宠物展开的面板里，**用量配额区**每个 Claude 账号前面现在是一个 emoji 人头 `👤`，
和整体像素风格不搭。请做一个**像素风的小头像图标**替换它。

### 规格
- **1 个文件**：`src/icons/account.svg`
- 一个通用的「用户头像」像素图标（**不针对具体某人**，泛用占位头像即可）：
  圆/方头 + 肩，或一个简洁的人形剪影，像素风、带描边。
- 同样 `viewBox 0 0 128 128`、透明底、无 `<text>`、深浅背景都清晰（这个显示在弹出菜单里，
  菜单背景通常较深，但仍请保证描边对比）。
- 显示尺寸约 **18px**（菜单行内小图标），所以**造型要极简**、小尺寸不糊。
- 建议配色与 Logo / 角色点缀语言协调（可用中性蓝灰或薄荷青系），别太花。

### 可选加分
- 若愿意，可再做一个 `account-stale.svg`（同头像但带「过期/离线」感，如灰一点或带个小时钟/感叹），
  用于账号用量数据过期时显示。不做也行，开发会沿用主头像。

## 三、其余可设计 / 优化点（清单，按需挑）

下面是通览扩展 UI 后整理的可设计点，**非必须**，design 觉得值得做的挑着做：

| # | 位置 | 现状 | 设计机会 |
|---|------|------|----------|
| 1 | **无会话待机** | 顶栏没有活跃会话时显示 emoji `😴` | 做一个像素「待机/睡觉中的小屋/notchi 标志」图标 `src/icons/idle-empty.svg`，让空闲时也是像素风（而非 emoji） |
| 2 | **用量进度条** | 文本字符 `▓▓▓░░ 60%` | （进阶）设计一套像素风进度条素材（如 0–10 格的像素电量条 SVG）。注意：需开发改渲染方式，成本较高，列为可选 |
| 3 | **弹出面板顶部** | 直接是会话列表，无标题/品牌 | 设计一个窄横幅/页眉素材（Logo 像素小图 + "Notchi" 像素字样轮廓），放在菜单顶部做品牌感 |
| 4 | **设置窗口** | 纯文字分组，无插画 | 设置页顶部可放 Logo 或一张「全角色合影」像素插画做门面（用于 README/扩展商店也好看） |
| 5 | **扩展商店配图** | 暂无 | 提交 extensions.gnome.org 需要截图/封面，可设计一张展示全角色 + 顶栏效果的像素风封面图 |
| 6 | **Logo 复用** | 已有 `notchi-logo.svg`，仅作仓库资产 | 若一轮 Logo 想随新区分度风格升级，可一并重做 |
